# Getting-a-Good-Night-s-Sleep
SleepInc provided anonymized user data from their sleep tracking app SleepScope, with the goal of uncovering how lifestyle factors influence sleep quality.  This project explores relationships between:  🏃 Physical activity 🧑‍💼 Occupation ⚖️ BMI category 😰 Stress levels 🛌 Sleep duration &amp; quality
